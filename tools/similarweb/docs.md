# Similarweb

## Agent Summary

Similarweb lets agents work with Websites, Apps, Traffic Metrics, Engagement Metrics, Keywords, Referrals, and Audiences through official MCP and API surfaces. Check auth, pagination, rate limits, source links, and lower write risk before using Similarweb in automations.

Agent readiness score: 5/5.

## Available Surfaces

- Official MCP: yes
- Official CLI: no
- Official API: yes
- OpenAPI/spec: unknown
- llms/AI docs: unknown
- Official SDK: unknown
- Community MCP: unknown
- Community CLI: unknown
- Community SDK / integration: unknown

## Auth

API keys are generated by account admins, must be activated, and are tied to account permissions plus user credit allowances.

## Main Objects

- Websites
- Apps
- Traffic Metrics
- Engagement Metrics
- Keywords
- Referrals
- Audiences
- Batch Reports
- Capabilities
- Credits
- Retail Intelligence
- MCP Queries

## Rate Limits

REST: 10 requests/second/account; Batch: 20 pending requests/user.

## Pagination

limit, offset, sort, asc on many aggregated endpoints.

## Agent Caveats

- Destructive action risk: low.
- Official MCP access and query limits depend on the customer's Similarweb subscription and LLM platform licensing.
- V5 API docs are current/beta in places; verify endpoint-level examples before building long-running syncs.
- Prefer official docs and SDKs first. Use community MCP/CLI/SDK sources only when clearly marked unofficial.
