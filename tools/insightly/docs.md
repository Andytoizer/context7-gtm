# Insightly

## Agent Summary

Insightly lets agents work with Contacts, Organizations, Opportunities, Leads, Projects, Tasks, and Events through official API surfaces plus community integration support. Check auth, pagination, rate limits, source links, and high write risk before using Insightly in automations.

Agent readiness score: 3/5.

## Agent Caveats
